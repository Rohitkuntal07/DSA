## Problem Statement
Given an array of integers `arr[]` and an integer `target`:

- **Variant 1:** Return **YES** if any two different numbers sum to the target, else **NO**.
- **Variant 2:** Return the **indices** of such numbers, else return `[-1, -1]`.

## Example 1
Input:
arr = {2, 6, 5, 8, 11}, target = 14  
Output:
YES  
[1, 3]

## Example 2
Input:
arr = {2, 6, 5, 8, 11}, target = 15  
Output:
NO  
[-1, -1]
