## Search an Element in a Linked List

### Problem Statement
Given the head of a linked list and an integer value, determine whether the value is present in the linked list or not.  
Return true if it is present, otherwise return false.

---

### Examples

Example 1

Input:  
0->1->2, val = 2

Output:  
True

Explanation:  
Element 2 is present in the list, so return true.

---

Example 2

Input:  
12->5->8->7, val = 6

Output:  
False

Explanation:  
Element 6 is not present in the list, so return false.
