## Check if Two Strings are Anagrams of Each Other

### Problem Statement
Given two strings, check whether the two strings are anagrams of each other or not.

An **anagram** means both strings contain the same characters with the same frequency, but possibly in a different order.

---

### Examples

Example 1

Input:  
CAT, ACT  

Output:  
true  

Explanation:  
Since the count of every letter in both strings is equal.

---

Example 2

Input:  
RULES, LESRT  

Output:  
false  

Explanation:  
Since the count of characters (like U and T) is not equal in both strings.
