## Longest Palindromic Substring

### Problem Statement
Given a string s, return the longest palindromic substring in s.

---

### Examples

Example 1

Input:  
s = "babad"

Output:  
"bab"

Explanation:  
It is the longest palindromic substring.  
"aba" is also a valid answer.

---

Example 2

Input:  
s = "cbbd"

Output:  
"bb"

Explanation:  
It is the longest palindromic substring.
